# Rapid Counterpoint API v1.05.07 Release Notes
Released March 10, 2026

## Update Multi-Company File Copy Script for RapidGo Label Printing

The `Util-Copy-Files-Multi-Company.ps1` script has been updated to support copying the required SQL file used to resolve a RapidGo label printing issue.

- This change ensures labels print in the same order they were scanned in RapidGo.
- The script now copies the necessary SQL file across companies to apply the fix consistently.
